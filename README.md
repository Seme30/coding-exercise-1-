# Multi-Round Points Game

A real-time multiplayer game built with NestJS (backend) and React (frontend) using WebSocket communication.

## Project Structure

```bash
multi-round-points-game/
├── backend/
├── frontend/
├── README.md
└── .gitignore
```

## Setup Instructions

### Backend Setup
```bash
cd backend
npm install
npm run start:dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
## Development
- Backend runs on: http://localhost:3000
- Frontend runs on: http://localhost:5173
